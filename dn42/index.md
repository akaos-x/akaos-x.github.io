# UNi-Network DN42


# 介绍
UNIFYZ-DN42 (AS4242421100) 网络是由 unify-z(听风) 全权运营的 DN42 网络。

该网络仅支持 WireGuard。

Looking Glass: https://lg.unifyz.top/

&gt; [!WARNING]
&gt; 本网络大部分节点（中国大陆节点除外）不与中国大陆的服务器 Peer，以避免可能的法律问题。
